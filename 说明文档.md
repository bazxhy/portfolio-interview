# 个人介绍网站 - 项目说明

## 1. 使用了哪些AI工具

本项目主要使用 **Claude Code**（Anthropic 的 AI 编程助手）完成开发，具体包括：
- **Claude Code**：负责整体代码生成、样式设计、交互逻辑编写
- OCR 工具（OCR.space API）：用于提取题目图片中的文字需求

## 2. AI帮助完成了哪些工作

AI 帮助完成了以下核心工作：
- ✅ **需求分析**：从图片中提取题目要求，拆解为清晰的任务列表
- ✅ **项目结构设计**：规划了单页网站的完整结构（导航、首屏、关于、技能、项目、联系、页脚）
- ✅ **HTML 代码生成**：生成了语义化、结构清晰的 HTML5 代码
- ✅ **CSS 样式设计**：设计了深色主题的现代化界面，包含渐变效果、动画、阴影等
- ✅ **JavaScript 交互**：实现了导航栏滚动效果、滚动动画、活跃状态切换等
- ✅ **响应式适配**：桌面端优化，同时兼顾不同尺寸屏幕
- ✅ **整体项目组织**：文件结构、代码注释、文档编写

## 3. 自己手动修改了哪些内容

AI 生成了绝大部分代码，我手动进行了以下调整：
- 🔧 **个人信息填充**：将模板中的姓名、邮箱、GitHub 等占位信息替换为真实信息
- 🔧 **技能标签调整**：根据实际技术栈调整技能标签内容
- 🔧 **项目经历替换**：将示例项目替换为真实的项目经历
- 🔧 **样式微调**：根据个人审美调整颜色、间距、字体大小等细节
- 🔧 **部署配置**：手动完成 GitHub Pages 的部署配置

## 4. 遇到的问题，以及如何通过AI解决

| 问题 | AI 解决方案 |
|------|------------|
| 题目是图片格式，无法直接读取文字 | 使用 OCR API（ocr.space）提取图片中的中文文字 |
| 需要快速构建美观的个人网站 | AI 生成了完整的深色主题设计，包含渐变、动画等现代化效果 |
| CSS 动画效果调试 | AI 提供了 Intersection Observer 实现滚动触发动画 |
| 响应式布局适配 | AI 使用了 CSS Grid + 媒体查询，确保桌面端完美呈现 |
