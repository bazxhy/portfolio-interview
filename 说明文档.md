# 个人介绍网站 - 项目说明

## 1. 使用了哪些AI工具

本项目主要使用 **Claude Code**（Anthropic 的 AI 编程助手）完成开发，具体包括：
- **Claude Code**：负责整体代码生成、样式设计、交互逻辑编写、GitHub Pages 部署
- **OCR.space API**：用于从题目图片中提取文字需求

## 2. AI帮助完成了哪些工作

AI 帮助完成了以下核心工作：
- ✅ **需求分析**：从图片中提取题目要求，拆解为清晰的任务列表
- ✅ **项目结构设计**：规划了单页网站（导航、首屏、关于、技能、项目、联系、页脚）
- ✅ **HTML 代码生成**：生成了语义化、结构清晰的 HTML5 代码，包含真实个人信息
- ✅ **CSS 样式设计**：设计了深色玻璃态（glassmorphism）主题，包含粒子背景、渐变、毛玻璃卡片、动画进度条等现代效果
- ✅ **JavaScript 交互**：实现了打字机效果、滚动触发动画、数字计数动画、技能条动画、导航高亮、移动端菜单等
- ✅ **部署上线**：通过 GitHub Pages 完成永久部署

## 3. 自己手动修改了哪些内容

AI 生成了代码框架，我手动进行了以下调整：
- 🔧 **个人信息填充**：填入真实姓名（关福博）、学校（新乡工程学院）、专业（信息工程）、联系方式
- 🔧 **项目经历替换**：替换为 GitHub 真实项目（BOSS直聘AI自动投递、AI面试助手），匹配 README 描述
- 🔧 **技能列表调整**：突出 AI 开发工具使用能力，调整技能标签与进度条数值
- 🔧 **设计审核**：确认整体审美风格符合个人预期

## 4. 遇到的问题，以及如何通过AI解决

| 问题 | AI 解决方案 |
|------|------------|
| 题目是图片格式，无法直接读取文字 | 使用 OCR.space API 提取中文文字 |
| 需要快速构建美观的个人网站 | AI 生成深色玻璃态设计，参考现代 portfolio 风格 |
| Netlify Drop 需要每次手动拖拽 | 改用 GitHub Pages + gh-pages 分支实现永久自动部署 |
| GitHub 命令行认证失败（VPN网络限制） | 创建 gh-pages 分支直接推送，GitHub 自动识别部署 |
| 页面内容单调，需要更丰富 | AI 添加粒子背景、打字机效果、计数动画、技能进度条等 |
